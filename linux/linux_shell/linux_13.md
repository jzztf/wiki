## for循环命令

语法：

```bash
for var in list
do
    <commands>
done
```

或者

```bash
for var in list; do <commands>
done
```

> shell 列表：由空格分开的字符串，`ls`命令的结果也是列表

for循环的几种方式：

- 读取列表的复杂值


- 从变量读取列表
- 从命令读取值
- 更改分隔符
- 使用通配符读取目录
