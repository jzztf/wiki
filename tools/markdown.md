# 常用markdown语法

## 链接

- [markdown-syntax](https://markdown-syntax.com/)

## 锚链接[↑ TOP]

- html原始码

```
<div align="right">[↑ TOP](#链接)</div>
```

示例：

<div align="right">[↑ TOP](#链接)</div>

## 上一节和下一节

- html原始码

```
<table>
<tr>
<td align="center">**[上一节](#!linux/linux_1.md)**</td>
<td align="center">**[下一节](#!linux/linux_3.md)**</td>
</tr>
</table>
```

示例：

<table>
<tr>
<td align="center">**[上一节](#!linux/linux_1.md)**</td>
<td align="center">**[下一节](#!linux/linux_3.md)**</td>
</tr>
</table>


## 表格

- html原始码

```
<table>
    <tr>
        <th rowspan="2">值班人员</th>
        <th>星期一</th>
        <th>星期二</th>
        <th>星期三</th>
    </tr>
    <tr>
        <td>李强</td>
        <td>张明</td>
        <td>王平</td>
    </tr>
</table>
```

示例：

<table>
    <tr>
        <th align="center" rowspan="2">值班人员</th>
        <th>星期一</th>
        <th>星期二</th>
        <th>星期三</th>
    </tr>
    <tr>
        <td>李强</td>
        <td>张明</td>
        <td>王平</td>
    </tr>
</table>


## 页面位置

- html原码

```
<center>居中</center>

<div align="center">居中</div> 
<div align="left">左对齐</div>
<div align="right">右对齐</div>
```

示例：

<center>居中</center>

<div align="center">居中</div>
<div align="left">左对齐</div>
<div align="right">右对齐</div>
