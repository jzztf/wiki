# python语句简介

- 重访python程序结构
  - 程序 <== 模块 <== 语句 <== 表达式 <== 对象
- Python语句

```python
赋值					=> 创建引用值
调用					=> 执行函数
打印调用				=> 打印对象 
if/elif/else			=> 选择动作
for/else				=> 序列迭代
while/else				=> 一般循环
pass					=> 空占位符
break					=> 循环退出
continue				=> 循环继续
def						=> 函数和方法
return					=> 函数和结果
yield					=> 生成器函数
global					=> 命名空间
nonlocal				=> 命名空间
import					=> 模块访问
from					=> 属性访问
class					=> 创建类
try/except/finally		=> 捕捉异常
raise					=> 触发异常
with/as					=> 环境管理器
del						=> 删除引用
```

- 两个if
  - `if (x > y) {command}`
